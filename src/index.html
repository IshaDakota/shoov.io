What is it?

Shoov is a tool for visual regression testing and live monitoring using your prefered functional testing framework (e.g. casperJS, Behat, etc.). You can read about it here:
  • Visual regression tests on every commit
  • How we could monitor Twitter (if we had to)
  • Cross Browser Visual Regression Tests With Shoov
  • Shoov - CI tests on the live site
Shoov and all its sub-components are open source!
